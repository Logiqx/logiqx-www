Welcome to www.logiqx.com - Source of CMPro/RomCenter data files and other useful tools.

To see who you have to thank for the contents of this site, have a look at the team page.

If you have questions about datfiles then please use Eric Bole Feysot's datfile message board.
Although I don't read it regularly, I do pass by every once in a while to see if there are any outstanding queries.

I have also provided my e-mail address but please pay careful attention to the list of things that I don't want to be e-mailed about.

Thankfully, the excessive interest CPS-2 games seems to have died down now but my CPS-2 questions and answers page is still online should you want to see what has been XORed, etc.
