Welcome to www.logiqx.com - Source of CMPro/RomCenter data files and other useful tools.

If you have questions about datfiles then please use Eric Bole Feysot's datfile message board.
Although I don't read it regularly, I do pass by every once in a while to see if there are any outstanding queries.
