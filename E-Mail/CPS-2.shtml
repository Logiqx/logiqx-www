Introduction

This document has been created because of the endless e-mails I receive about CPS-2 emulation. I have no special interest in the CPS-2 games so I would like to keep everyone away from my e-mail box and find the answers for themselves. Please read this document carefully and if your question isn't answered then search elsewhere on the web. All CPS-2 related e-mail I receive is now sent straight to my 'deleted items' and will get no response.

CPS-2 game list

This is a list of all CPS-2 games that have been dumped and are now available on the web (games in grey are unreleased though). In addition to being dumped the games also need decrypting before they can be be emulated. As soon as a new game is decrypted you can guarantee there will be support in CPS2MAME, Kawaks and Nebula. There are other CPS-2 games not on this list but they have not even been dumped yet. Do not ask me about any games not on this list.

Where do I get the ROMs?

Use a search engine. Don't ask me where they are as I don't know.

What emulators support CPS-2 games?

There are several CPS-2 emulators available:

CPS2MAME - A CPS-2 only version of MAME for DOS, produced by CPS2Shock. There are other variants available for DOS, Windows and Linux.

Kawaks - My favorite CPS-1 and CPS-2 emulator.

Nebula - A good CPS-2 emulator for Windows and can have new games added quickly and easily using text drivers.

Final Burn Alpha  - Another good CPS-2 (and Sega) emulator for Windows that is being maintained by the Final Burn Alpha team.

Calice - The CPS-2 emulator for Windows by David Raingeard. Another emulator where drivers are text based but they are more complicated than in Nebula.

ACE - The newest CPS-2 emu on the block but doesn't yet support all of the games.

How do I use the emulators?

Each emulator comes with a readme.txt file. If you read the file (novel idea?) it will explain how to use the emulator. If you still have problems getting the emulator working then ask for help in the CPS-2 board at Retrogames.

Are the CMPro and ROMCenter dats correct?

The CMPro and ROMCenter dats I produce contain the latest details known by CPS2Shock and MAME team. Raz and myself are very thorough at checking them so I'd appreciate it if you don't query the contents.

What is the best source of CPS-2 information?

CPS2Shock is the site responsible for decrypting the CPS-2 games. I suggest you visit it every now and then for the latest CPS-2 news. Do not bother Razoola because if he gets fed up then your games will not be decrypted for a very long time (despite what some people might think).
